# 基金净值估算看板

一个本地运行的轻量查询工具，用来跟踪基金盘中估值参考。

## 功能

- 输入 6 位基金代码加入自选
- 交易时段每 20 秒自动刷新估值，非交易时段每 5 分钟刷新
- 显示估算净值、估算涨跌幅、上一日单位净值
- 为每只基金绘制当日实时估值变化曲线
- 横轴按 A 股交易时段固定显示 `09:30 / 11:30 / 13:00 / 15:00`
- 本地保存自选列表

## 运行

```bash
python3 server.py
```

默认地址：

```text
http://127.0.0.1:8765
```

也可以指定端口：

```bash
python3 server.py 8788
```

## 说明

- 数据源使用公开可访问的基金估值接口，稳定性取决于上游。
- 页面展示的是盘中估值参考，不是基金公司最终披露净值。
- 自选列表保存在 `data/watchlist.json`。
- 曲线历史保存在浏览器本地存储里，按交易日滚动重置。
- 曲线是本地按刷新频率采样积累的分时参考，不是上游提供的完整逐笔分时数据。
